# 💰 SpendSense — AI Expense Tracker

 AI-powered expense tracker built with React, Tailwind CSS, Recharts, and Claude AI.

## ✨ Key Features

| Feature | Description |
|---|---|
| **AI Auto-Categorization** | Blur a description field → Claude API classifies it instantly |
| **AI Spend Insights** | Claude generates 3 personalized, ₹-specific financial insights |
| **Smart Budget Alerts** | Reactive alerts at 75% and 90% budget with actionable suggestions |
| **Recharts Visualizations** | Donut breakdown + monthly trend bar chart with custom tooltips |
| **Animated Count-Up** | Stats animate from 0 with ease-out cubic on mount |
| **Category Filtering** | Filter expenses by category with color-coded pills |
| **Budget Presets** | Quick-set budget with one-click presets |
| **Month-over-Month** | Spending comparison vs last month and 3-month average |

---

## 🚀 Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

---

## 🛠️ Tech Stack

- **React 18** — functional components + hooks
- **Tailwind CSS 3** — utility-first styling, zero inline CSS
- **Recharts 2** — PieChart (donut) + BarChart with custom tooltips
- **Vite** — fast dev server + build tool
- **Anthropic Claude API** — expense categorization + spend insights
